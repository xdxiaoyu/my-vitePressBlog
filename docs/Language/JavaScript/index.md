# JavaScript 学习

JavaScript 是前端开发的核心语言，这里收录了 JavaScript 及其各个版本的新特性学习笔记。

## 基础内容

### [JavaScript 基础](/Language/JavaScript/JavaScript/)
JavaScript 基础语法和核心概念，包括变量、函数、对象、原型链等核心知识。

### [ES6](/Language/JavaScript/ES6/)
ECMAScript 6 (ES2015) 新特性，包括 let/const、箭头函数、Promise、Class 等重要特性。

### [ES7-10](/Language/JavaScript/ES7-10/)
ES7 到 ES10 的新特性，包括 async/await、可选链、空值合并等实用特性。
