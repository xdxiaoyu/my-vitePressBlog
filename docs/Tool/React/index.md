# React 生态

React 是一个用于构建用户界面的 JavaScript 库，这里收录了 React 及其生态系统的学习笔记。

## 学习内容

### [React 基础](/Tool/React/React_base/)
React 基础知识和用法，包括 JSX 语法、组件、Props、State、生命周期等核心概念。

### [React Hooks](/Tool/React/React_Hooks/)
React Hooks 的使用方法和最佳实践，包括 useState、useEffect、useContext、自定义 Hooks 等。

### [React 核心原理](/Tool/React/React_Principle/)
深入剖析 React 的底层实现原理，从源码角度理解虚拟 DOM、Diff 算法、Fiber 架构、事件系统等核心机制。

### [React Router](/Tool/React/React_Router/)
React 官方路由库，包括路由配置、导航、路由守卫、动态路由等使用方法。

### [Redux 状态管理](/Tool/React/Redux/)
React 应用的状态管理解决方案，包括 Redux 核心概念、中间件、Redux Toolkit 等。
