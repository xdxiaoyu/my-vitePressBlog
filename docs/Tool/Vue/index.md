# Vue 生态

Vue.js 是一套用于构建用户界面的渐进式框架，这里收录了 Vue 及其生态系统的学习笔记。

## 学习内容

### [Vue3 核心特性](/Tool/Vue/Vue3/)
Vue3 的核心特性与实战，包括 Composition API、响应式系统、TypeScript 支持、性能优化等现代化开发实践。

### [Vue3 核心原理深度解析](/Tool/Vue/Vue3_Principle/)
深入剖析 Vue3 的底层实现原理，从源码角度理解响应式系统、编译器、虚拟 DOM、Diff 算法等核心机制。

### [Vue2 基础](/Tool/Vue/Vue_base/)
Vue2.js 基础知识和用法，包括模板语法、组件、指令、生命周期等核心概念。

### [Vue MVVM 原理](/Tool/Vue/Vue_mvvm/)
深入理解 Vue2 的 MVVM 架构原理，包括响应式系统、虚拟 DOM、双向数据绑定等实现原理。

### [Vuex 状态管理](/Tool/Vue/Vuex/)
Vue 官方的状态管理库，包括 State、Getter、Mutation、Action 等核心概念和使用方法。
