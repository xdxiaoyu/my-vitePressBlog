## ⚠️ 可能的问题

### 1. 外部图片加载失败

**位置**：`docs/Tool/Axios/index.md` 第527行

```markdown
![](https://img2020.cnblogs.com/blog/1639676/202008/1639676-20200805190425070-67870031.png)
```

**问题**：外部链接可能失效或加载缓慢

**解决方案**：
- 如果图片重要，建议下载到 public 目录
- 或者使用图床服务（如 GitHub、七牛云等）

### 2. 浏览器缓存问题

如果图片路径正确但仍显示不出来：

1. **清除浏览器缓存**
   - Chrome/Edge: `Ctrl+Shift+Delete` (Win) 或 `Cmd+Shift+Delete` (Mac)
   - 或者硬刷新：`Ctrl+F5` (Win) 或 `Cmd+Shift+R` (Mac)

2. **重启开发服务器**
   ```bash
   # 停止服务器：Ctrl+C
   # 重新启动
   pnpm dev
   ```

### 3. 开发服务器未检测到 public 目录

确保项目结构正确：

```
vitePressBlog/
├── .vitepress/
├── docs/
└── public/      ← public 目录应该在这里
    ├── bg.jpg
    ├── bg1.png
    └── ...
```

## 🐛 排查步骤

如果图片仍然无法加载，按以下顺序检查：

1. ✅ 确认图片文件在 `public/` 目录
2. ✅ 确认路径以 `/` 开头（如 `/bg.jpg`）
3. ✅ 直接访问图片 URL（如 `http://localhost:5173/bg.jpg`）
4. ✅ 检查浏览器控制台是否有错误信息（按 F12）
5. ✅ 清除浏览器缓存并硬刷新
6. ✅ 重启开发服务器
