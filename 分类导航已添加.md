## 🎯 已添加的功能

### 1. 顶部导航栏
页面顶部始终显示导航栏（在所有页面都可见）：
- 🏠 **首页** - 回到主页
- 📚 **语言** - 查看所有语言相关文档
- 🛠️ **工具** - 查看所有工具相关文档

### 2. 首页优化
首页现在有多个分类入口：

#### 主要按钮
- 📚 **语言学习** - 进入语言分类页
- 🛠️ **工具学习** - 进入工具分类页

#### 特色卡片（6个）
每个卡片都可以直接点击进入对应文档：
1. 📝 **JavaScript** - JavaScript、ES6、ES7-10
2. 🚀 **Node.js** - Node、MongoDB、爬虫
3. 💚 **Vue 生态** - Vue、MVVM、Vuex
4. 📦 **构建工具** - Webpack
5. 🌐 **HTTP 工具** - Axios
6. 🔧 **开发工具** - Git、SSO

### 3. 分类索引页
新增了两个分类索引页面：

#### `/Language/` - 语言学习页面
列出所有语言相关的文档：
- JavaScript 基础
- ES6
- ES7-10
- Node 基础
- MongoDB
- 爬虫

#### `/Tool/` - 工具学习页面
列出所有工具相关的文档：
- Vue 基础
- Vue MVVM
- Vuex
- Webpack
- Axios
- Git
- SSO

### 4. 侧边栏导航
进入任何文档页面后，左侧会显示当前分类的侧边栏：
- 在语言类文档中，显示 JavaScript 和 Node 的所有文章
- 在工具类文档中，显示 Vue 和其他工具的所有文章

## 🚀 如何使用

### 方式 1：顶部导航栏
1. 点击顶部的 **语言** 或 **工具**
2. 进入对应的分类页面
3. 选择你想看的文档

### 方式 2：首页按钮
1. 在首页点击 **语言学习** 或 **工具学习** 按钮
2. 进入对应的分类页面
3. 选择你想看的文档

### 方式 3：首页卡片
1. 在首页直接点击感兴趣的卡片
2. 直接进入对应的文档

### 方式 4：侧边栏
1. 进入任何文档后
2. 使用左侧侧边栏快速切换到同类别的其他文档

## 📱 响应式设计

在移动设备上：
- 顶部会有汉堡菜单按钮 ☰
- 点击可展开导航菜单
- 侧边栏也可以通过菜单按钮访问
